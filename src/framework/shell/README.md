# Shell

The shell defines the persistent page layout that wraps all product experiences.

## Layout Zones

| Zone | Description |
|------|-------------|
| **Top Bar** | Global header with search, profile, and context actions |
| **Sidebar** | Navigation panel -- content changes based on nav state and lifecycle phase |
| **Content Area** | Main region where treatments and product screens render |
| **Expansion Panel** | Slide-out panel for contextual detail, previews, and secondary flows |
