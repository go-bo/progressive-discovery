# Templates

Full-page templates that combine patterns and components for specific discovery moments in the user journey.

| Template | Purpose |
|----------|---------|
| [UnpurchasedSkuPage](./UnpurchasedSkuPage/) | Page shown when a user navigates to a product they haven't bought |
| [PlatformEmptyState](./PlatformEmptyState/) | Minimal empty states for platform surfaces the user hasn't engaged with |
| [FeatureActivation](./FeatureActivation/) | Introduce a capability and drive the user to activate it |
