# Feature Activation Template

Introduces a specific capability within a purchased product and drives the user to activate it. Used for features that benefit from guided onboarding rather than organic discovery.

## Pattern

The page follows a structured composition:

1. **Value Framing** — Why this feature matters, framed around the user's situation
2. **Template Previews** — Visual previews of pre-built templates or configurations the user can start from
3. **Primary CTA** — "Start with template" (or equivalent) to launch a guided setup flow
4. **Secondary CTA** — "Start from scratch" or "Learn more" for users who want more control

## Where Used

- Time: Policy setup, schedule templates
- Spend: Expense policy setup, card controls
- Platform: Workflow templates, report templates

## Components / Patterns Used

- PageHeroBanner, FeatureCard, DiscoveryPageLayout
