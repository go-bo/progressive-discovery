# Unpurchased SKU Page Template

Full-page template shown when a user navigates to a product they haven't purchased. Communicates product value and drives consideration without being pushy.

## Pattern

The page follows a structured composition:

1. **Hero** — Product name, tagline, and key visual
2. **Value Proposition** — Core benefits framed around the user's current context
3. **Readiness Card** — Shows what the user already has in place that makes this product easy to adopt (e.g., "You already have 12 employees set up")
4. **Discover More Products** — Cross-sell surface showing other available products

## Where Used

- Time product (for Spend trial users)
- Spend product (for Time trial users)
- Any product the user hasn't purchased when navigating via global home

## Components / Patterns Used

- PageHeroBanner, FeatureCard, DiscoveryPageLayout
