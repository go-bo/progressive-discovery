# Nav/Home Sketch

A future vision exploration for pane-based navigation.

## Concept

Explores an alternative navigation model where the sidebar is replaced by a layered pane system:

- **Basement** -- Persistent bottom layer with global navigation and product switching
- **PaneA** -- Primary content pane that slides over the Basement, containing product-specific navigation and content

This model trades the traditional sidebar for a more spatial, depth-based navigation metaphor.
