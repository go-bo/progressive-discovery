# Explorations

Self-contained future vision prototypes that push beyond the current framework.

## What These Are

Explorations are standalone demos that investigate new interaction models, navigation paradigms, or product concepts. They may import components from the framework but maintain their own layout and routing -- they are not constrained by the current framework architecture.

## Relationship to Framework

Explorations can pull from the framework but are not required to use it. Successful explorations may eventually influence or be absorbed into the framework. Failed explorations are preserved as learning artifacts.
